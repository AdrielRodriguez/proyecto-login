# Login App

Este proyecto es una **aplicación de login simple** creada con React y Vite.  
Permite ingresar email y contraseña y muestra los datos en una alerta al enviar el formulario.

## Tecnologías utilizadas

- React
- Vite
- JavaScript
- HTML & CSS
